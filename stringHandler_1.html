Qusetion: Given two strings, if they have same types of characters, and for each kind of character, they have the same number, then we call that the two strings are matched.
For example, "abbdcca" and "bdcaacbq" are matched.

Algorithm idea: Use a 256 length integer array to store the number of times of each character. Then traverse string1 and string2 from their first character, for string1, the flag of each type of character would do a plus-one operation and for string2, the flag would do a minus-one operation. And if the two strings are matched, the integer array will be an array with all 0 value elements.

Code: 

/**
 * Given two strings, if they have same types of characters, 
 * and for each kind of character, they have the same number, 
 * then we call that the two strings are matched.
 * For example, "abbdcca" and "bdcaacbq" are matched.
 * */
package string_handler;

public class StringMatcher {

    public static void main(String[] args) {
        String s1 = "abbdcca213212!@@#@";
        String s2 = "bdcaacb212312@@@#!";

        System.out.println(StringMatcher.judgeMatch(s1, s2));

    }

    /**
     * Given two strings, judge whether they are matched.
     * 
     * @param s1
     * @param s2
     * @return true or false
     * */
    private static boolean judgeMatch(String s1, String s2) {
        if (null == s1 || null == s2) {
            return false;
        }

        if (s1.equals(s2)) {
            return true;
        }

        if (s1.length() != s2.length()) {
            return false;
        }

        int[] flag = new int[256];
        for (int i = 0; i < s1.length(); i++) {
            char temp1 = s1.charAt(i);
            char temp2 = s2.charAt(i);
            flag[temp1]++;
            flag[temp2]--;
        }

        for (int i = 0; i < flag.length; i++) {
            if (flag[i] != 0) {
                return false;
            }
        }

        return true;
    }
}
