Let's go to the topic directly!

Quetion: Remove invalid elements in a vector with all integer elements. These invalid elements are integers which are odd numbers.

Algorithm idea: Traverse the vertor from both the start and the end. Find the first invalid element <a> from the start and find the first valid element <b> from the end, then change the value of <a> into the value of <b>.
Code talks:


package word_ladder;

import java.util.Vector;

public class DelElementInVector {

    public static void main(String[] args) {

        Vector<Integer> vector = new Vector<Integer>();// initialize a vector

        /* add ten elements in the vector */
        for (int i = 0; i < 100; i++) {
            vector.add(i);
        }

        System.out.println("Before removing: ");
        showVector(vector);// print the vector before removing these illegal
                            // elements

        int start = 0;
        int end = vector.size() - 1;
        while (start < end) {
            while (vector.get(start) % 2 == 0 && start < end) {
                start++;
            }

            while (vector.get(end) % 2 != 0 && start < end) {
                vector.remove(end);
                end--;
            }

            vector.set(start, vector.get(end));
            vector.remove(end--);
        }

        System.out.println("After removing: ");
        showVector(vector);// print the vector after removing these illegal
                            // elements

    }

    /**
     * Print the elements in the vector
     * 
     * @param vector
     * @return
     * */
    private static void showVector(Vector<Integer> vector) {
        for (int i = 0; i < vector.size(); i++) {
            System.out.println(vector.get(i));
        }
    }
}

Note: Performance should be took into consideration. You cannot just remove one by one because the size of vector is changeable.
